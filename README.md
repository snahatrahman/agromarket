# 🌾 AgroMarket

কৃষক থেকে ভোক্তা মার্কেটপ্লেস - Production Ready Version

## 🚀 Quick Start

1. VS Code দিয়ে এই folder খুলুন
2. Live Server extension install করুন
3. index.html এ right click → "Open with Live Server"
4. Browser খুলবে - http://127.0.0.1:5500

## 💾 Data Storage

বর্তমানে LocalStorage ব্যবহার হচ্ছে। Backend যোগ করার সময় এই keys:

- `agromarket_products` - Products
- `agromarket_users` - Users
- `agromarket_orders` - Orders
- `agromarket_cart` - Cart items
- `agromarket_current_user` - Current logged in user
