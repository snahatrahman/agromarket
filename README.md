# 🌾 AgroMarket


AgroMarket is a web-based agricultural marketplace platform that connects farmers directly with consumers. The system allows farmers to manage products and consumers to browse, purchase, and review agricultural products through an easy-to-use interface.

## Project Overview

The goal of AgroMarket is to reduce the gap between farmers and consumers by providing a centralized online marketplace where agricultural products can be bought and sold efficiently.

## Features

### Consumer Features

* User Registration and Login
* Browse Available Products
* Product Details View
* Shopping Cart Management
* Checkout System
* Order History Tracking
* Product Reviews

### Farmer Features

* Farmer Dashboard
* Add New Products
* Manage Existing Products
* Product Inventory Management

### Admin Features

* Admin Dashboard
* Product Monitoring
* User Management
* Order Management

## Installation

### Requirements

* XAMPP
* Apache
* MySQL
* PHP

### Setup Steps

1. Clone the repository

git clone https://github.com/snahatrahman/agromarket.git

2. Move the project into the XAMPP htdocs folder

3. Start Apache and MySQL from XAMPP Control Panel

4. Import the database using phpMyAdmin

5. Open the application in your browser

http://localhost/agromarket
